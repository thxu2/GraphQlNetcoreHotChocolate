Есть структура данных, лежащая в файле sample-catalog.json.
Есть описание этой структуры: catalog-api.pdf.

Нужно: создать GraphQL сервис на .net core, который будет предоставлять доступ к данным sample-catalog.json.
Для представления в GraphQL для верхнего уровня нужно использовать содержимое массива courses (сущность course).

Для запроса
{
  course(id: 2) {
    title
  }
}

ожидается следующий ответ:
{
  "data": {
    "course": {
      "title": "Основы написания примеров"
    }
  }
}
